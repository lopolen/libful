# Статуси примірників книг

Поле `book_copies.status` наразі навмисно захардкоджене.

Дозволені значення:

- `available`
- `borrowed`
- `lost`
- `damaged`

Це спрощення для ранньої версії проєкту. Воно допомагає тримати доменну логіку меншою та не створювати окрему таблицю статусів до того, як правила стануть складнішими.

Важливо: якщо цей список зміниться, потрібно оновити всі місця, які від нього залежать:

- валідацію в застосунку
- бізнес-логіку
- API-документацію
- тести
- міграції бази даних або check constraints, якщо вони будуть додані пізніше
